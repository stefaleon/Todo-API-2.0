## Todo API 2.0
* RESTful API that can CRUD "to do" notes
* MongoDB collections models: Todo, User
* no front-end available, check with Postman or equivalent

### dependencies
* mongoose
* Express
* body-parser
* validator -> email validation
* jsonwebtoken -> user authentication

### dev-dependencies
* nodemon
* Mocha
* expect
* supertest
